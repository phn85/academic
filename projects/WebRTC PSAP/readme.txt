The step to use this project:
1.install node.js
2.open the cmd in windows or terminal in linux
3.cd to the project folder and type "node index.js 5000" 5000 is the port number
4.go to the chrome and type the link 'http://localhost:5000/index.html'
5.open 2 new tabs and copy and paste the link in index page, then you can see the caller and psap pages.
